# Descrição

O repositório trata-se da realização da atividade PC.03 da matéria de desenvolvimento móvel, na qual realizou-se alterações no programa de todolist passado pelo professor com o intuito de:
* Permitir a implementação da funcionalidade de definir a prioridade de uma tarefa
* Permitir o aumento do grau de prioridade de tarefa pelo seu menu de seleção com a inserção da opção de prioritize
